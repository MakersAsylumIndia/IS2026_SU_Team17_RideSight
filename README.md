# IS2026 Spring - RideSight

Problem Statement-
Motorcycle/Scooter drivers, especially those working for delivery companies, rely heavily on smartphone navigation to complete their journeys. However, checking directions requires them to repeatedly look down at their phones, diverting their attention away from the road. As the number of delivery drivers continues to grow, there is an increased likelihood of them getting into a severe accident, creating an urgent need for a safer, hands-free navigation solution that allows riders to receive directions in real-time.

Solution-
We are developing a smart navigation attachment for a motorcycle helmet with an embedded ESP32 and two RGB LEDs: one on the left and one on the right. The ESP32 will be connected wirelessly over bluetooth to a navigation application on the user's phone. The navigation application will provide real-time instructions, which will be transmitted to the ESP32. The ESP32 will then control the LED lights placed on both sides of the helmet. For example, if the user has to go right, the LED light on the right side of the helmet will blink.

Future Prospects-
This project could be further developed by integrating additional navigation cues and expanding compatibility with different navigation applications. In the future, the system could be adopted by delivery companies to improve rider safety and reduce accidents caused by distracted driving.

Contents of all the folders-
CAD contains all of the files of the components which we 3D printed/laser cut for our project.
Code contains the codes for our project.
Documentation contains the steps to making this project.
Electronics contains the circuit diagram with all the components which we used for our project.
Photos_Videos contains photos and videos of the working prototype.
Reference_Data contains all the references that we used.
BOM.csv contains all the electrical components that we used.
