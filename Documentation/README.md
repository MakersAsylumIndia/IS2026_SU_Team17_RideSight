We first used Mapbox API and designed a webapp for our navigation system. Then we linked our ESP 32 to this webapp using bluetooth. After finalising the circuit and the components, we designed and 3D printed a box to fit all of the components. This box is supposed to be attached to the helmet using a sliding mechanism which we designed and 3D printed. Finally we mounted the LEDs on the helmet and connected them to the box. The box is detachable so that the user the charge the Li-Po battery inside which powers the entire circuit. The user just has to put the location of where they want to go to on the webapp and the LEDs will indicate direction accordingly.
